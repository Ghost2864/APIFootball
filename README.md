# APIFootball

Простой Python API клиент для API-FOOTBALL с кэшированием через Redis.

## Особенности
- Получение информации о командах, игроках и лигах
- Кэширование запросов с использованием Redis

## Установка
```bash:
git clone https://github.com/<>/APIFootball.git
cd APIFootball
poetry install


python main.py
